Тихий городской ветрогенератор

Проект механизма будущего для экологически чистых городов

Введение

Современные города потребляют огромное количество электроэнергии. Большая часть этой энергии по-прежнему производится с использованием ископаемого топлива, что приводит к загрязнению окружающей среды и изменению климата.

Цель данного проекта — предложить механизм, который позволит получать чистую энергию прямо в городской среде, без вреда для людей и природы.

Проблема существующих ветряков

  • Создают высокий уровень шума
  • Опасны для птиц
  • Требуют больших открытых пространств
  • Плохо работают при слабом ветре

Из-за этого традиционные ветряные турбины редко устанавливаются непосредственно в городах.

Предлагаемое решение

Тихий городской ветрогенератор — это компактный вертикальный ветрогенератор нового типа, специально адаптированный для установки в жилых районах.

Устройство механизма

  • Вертикальный ротор
  • Аэродинамический корпус
  • Тихоходный генератор
  • Контроллер управления
  • Аккумуляторный блок
